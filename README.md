# Nike Air Max Pulse - Scrollytelling Landing Page

A premium, Awwwards-caliber Nike landing page featuring an immersive **cheetah scrollytelling intro** powered by a dual-canvas rendering engine.

## Features

- **Dual-Canvas Tiger Engine** - 242-frame cheetah animation controlled by scroll
- **Ambient Fill** - Blurred, saturated background canvas eliminates black bars on all screens
- **Buttery Smooth Scroll** - Lenis + GSAP ScrollTrigger integration
- **Responsive Design** - Mobile-first, pixel-perfect on all viewports
- **Premium UI** - Editorial typography, micro-animations, reveal transitions

## Tech Stack

- HTML5 (Semantic)
- CSS3 (Custom Properties, Flexbox, Grid)
- JavaScript ES6+ (Vanilla, Class-based)
- [GSAP](https://greensock.com/gsap/) with ScrollTrigger
- [Lenis](https://github.com/studio-freight/lenis) Smooth Scroll

## Sections

1. **Hero** - Full-screen sticky scrollytelling cheetah animation (800vh)
2. **First Look** - Nike Air Max Pulse product introduction
3. **Best of Air Max** - Horizontal snap carousel with product cards
4. **Cinematic Banner** - 85vh cheetah forest feature image
5. **The Essentials** - 3-column Men's / Women's / Kids' lifestyle grid
6. **Footer** - 4-column layout with social links

## Getting Started

npx serve .

# Open in browser
http://localhost:3000
